## Song Recommendation System Using AI/ML 

## Project Overview

This project is a **Machine Learning based Song Recommendation System** that suggests songs to users based on similarity between songs and their features. The system analyzes song data and recommends similar tracks using Python and data science techniques.

## Features

* Recommends songs based on similarity
* Uses machine learning algorithms
* Data preprocessing and analysis
* Implemented using Python and Jupyter Notebook
* Beginner-friendly AI/ML project

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook
* kaggle

## Working of the Project

1. Load the dataset containing song information.
2. Perform data cleaning and preprocessing.
3. Extract important features from the dataset.
4. Apply similarity algorithms to find related songs.
5. Recommend songs based on similarity score.

## How to Run the Project

1. Clone the repository.
2. Install required Python libraries.
3. Open the project in Jupyter Notebook.
4. Run the notebook cells step by step.
5. Enter a song name to get recommendations.

## Project Structure

* dataset – Contains song dataset
* notebook – Machine learning model implementation
* README.md – Project documentation

## Output

The system recommends a list of songs that are similar to the selected song based on the dataset and machine learning algorithm.

Developed as part of a **Machine Learning learning project**.
